INDIGO Python’s documentation¶

On this site let’s talk about a implementation of a library INDIGO in Python. You can learn more about original INDIGO and how it works.

You can see the source code in GitHub to understand all you need to use this library. The repository is composed of two folders:

  • project: Here it is the source code of the library, as well as an example of a small program that works as an INDIGO client.
  • docs: This section contains all the files required to produce this documentation.

The documentation will have the following structure:

  • 1. INDIGO_Client
    • 1.1. INDIGODevice
    • 1.2. INDIGOElement
    • 1.3. INDIGOProperty
    • 1.4. INDIGOServerConnection
