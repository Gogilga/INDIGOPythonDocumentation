2. INDIGO_Client_Example¶

Here, you can see the functions that take part in the example of a client of INDIGO.

connected(property)[source]¶

In this function, we add a listener to display a message when the device in question is switched on or off.

Parameters:property (INDIGOProperty) – A property to include the listener.
connection()[source]¶

This is a listener that is called if we lose the connection with the server. It shows a message for this reason and stops the execution of the program.

program()[source]¶

This is the main part of the code. Here, there is an implementation of a small client to connect to an INDIGO server. First of all, you must enter a host IP address and port to connect. Lately, a small menu is displayed and you can navigate it by typing the number of the option you want to select.

You can see and download the source code here.
